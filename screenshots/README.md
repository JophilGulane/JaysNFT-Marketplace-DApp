# Screenshots Directory

Screenshot images are stored here with the following naming convention:

- `home-page.png` - Landing/Home page hero section
- `explore-collections.png` - Explore Famous Collections section from home page
- `how-it-works.png` - How It Works section from home page
- `mint-page.png` - Mint NFT page screenshot
- `my-nfts-page.png` - My NFTs page screenshot
- `marketplace-page.png` - Marketplace page screenshot
- `activity-page.png` - Activity feed page screenshot
- `admin-page.png` - Admin panel screenshot
- `footer.png` - Footer section screenshot

The images are automatically referenced in the main README.md file.
